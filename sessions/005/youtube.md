## Description (paste verbatim into YouTube)

```
Three AI builders got the same live audience brief: build a bakery business. No rehearsal, no pre-built demo, no hidden work. In under an hour, they turned that prompt into three different prototypes while sharing their screens, tools, prompts, mistakes, and decision-making.

Justin Novak built Milk and Cookies, a marketplace that connects local farmers, ghost-kitchen bakers, delivery drivers, and customers. He used Wispr Flow to talk through the product brief, Midjourney and Figma for the first brand pass, Claude Design for a cleaner visual identity, and Claude Code inside VS Code to build the marketplace flow.

Alec Saluga built Crumb B2B, a sales and retention tool for sending logo-printed cookies to prospects or clients. He used FireShot to capture Crumbl as a visual reference, Claude and ChatGPT for ideation, Lovable to build the app, Higgsfield for logos and AI UGC, and Gemini/Nano Banana for the personalized cookie mockup concept.

Ian Kilpatrick built Sunday Crumb, a Nashville bakery truck where customers scan a QR code, create their own pastry, choose fillings and toppings, name it, and submit it to the truck. He ran ChatGPT plus multiple OpenClaw agents in parallel to build the brand kit, landing page, order flow, and admin view.

This is AI4NTP, a free live series for non-technical operators learning to use and build with AI. No theory, no hype. Real operators show the work so you can copy the workflow yourself. Translators, not oracles.
```
