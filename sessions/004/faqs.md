# Episode 004 FAQ

### How do I know that what the AI delivered is actually valid?

This came up live (from a live attendee, Rolando). The short answer is that you stay in the loop and verify the output, you do not blindly trust it. This is exactly why Alec walked through n8n during the session: in a visual workflow you can see every step, find where data is leaking or broken, and check the result against reality. Treat AI output like work from a new hire. You review it before you ship it.

### When should I use n8n versus Claude Code?

A live attendee (David Toledo) asked whether Alec still recommends n8n over Claude Code because the debugging is more visual. Alec said it is case by case. If you have never built these workflows or you are not technical, n8n is a great starting point because it is visual and teaches you the fundamentals (what a system prompt is, how to make an HTTP request, where things break). Once you are comfortable, use Claude Code for anything complex and reach for n8n when you just need to wire a few tools together quickly.

### Wispr Flow or Super Whisper, which one should I use?

Another question from a live attendee (David Toledo), who uses Super Whisper and was hesitant to switch. Alec's honest answer: he has not used Super Whisper, but he assumes it does the same thing (press a hotkey, talk, and it transcribes). His rule of thumb is simple. If Super Whisper is working well for you, there is no reason to switch. If you are running into problems, Wispr Flow has been great for him.

### Are these scrapers (like Apify) against terms of service?

A live attendee (Erika) raised this during the Apify demo, and we want to be straight with you: we did not fully address it on the call, so here is a careful answer. Scraping publicly available data sits in a genuinely gray area legally, and every platform has its own terms of service that you are responsible for reading and respecting. Some sites prohibit scraping outright, others allow it within limits, so check each platform's terms before you run anything and do not collect private or personal data you are not entitled to. This is not legal advice. If you are operating at any real scale, talk to a lawyer.

### Big picture: Hermes versus Claude Code, when do I use which?

A live attendee (Jayme) asked for the 50,000-foot view, and both hosts answered. Use Hermes (or OpenClaw) for anything scheduled or always-on: cron jobs, recurring marketing tasks, persistent memory that remembers everything about you. Use Claude Code as your daily driver, the place you go to sit down and build something hands-on with your environment and API keys set up. Alec put it simply: if it runs all the time, use Hermes or OpenClaw; if you are going in to get focused work done, use Claude Code.
