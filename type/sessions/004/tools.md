# Episode 004: tools and resources mentioned

"7 AI tools you've never heard of," aired June 18, 2026. Hosts Alec Saluga and Justin Novak, with guest James Biernesser (founder, Dartee Golf).

Every tool, app, company, model, and resource named on the call, grouped by who presented it.

## Alec's tools

**[Codex](/ai-tools/codex)** — OpenAI's coding agent (think of it as their version of Claude Code) that writes and runs code, so Alec used it to hit the SpyFu API and build a prospect list in about 10 minutes instead of days. *Demoed by Alec.*

**[Apify](/ai-tools/apify)** — A library of ready-made web scrapers for pulling data off sites like Instagram, YouTube, LinkedIn, and Facebook, which you can run on their platform or hand to an agent. *Demoed by Alec.*

**[n8n](https://n8n.io)** — A visual, open-source workflow builder for wiring tools together and routing data, self-hostable for a few dollars a month with unlimited executions. *Demoed by Alec.*

**[PLAUD Note](/ai-tools/plaud-note)** — A pocket-sized external recording device that captures meetings, calls, and in-person conversations, then pipes them into your other tools (Alec routes his into Google Drive via Zapier). *Demoed by Alec.*

**[Higgsfield](https://higgsfield.ai)** — An aggregator of image and video generation models, plus a marketing studio for UGC content, that lets you run one prompt through several models and compare outputs. *Demoed by Alec (honorable mention).*

**[SpyFu](https://www.spyfu.com)** — A competitive-intelligence service with an API for pulling current advertisers by keyword, which Alec fed to Codex to build his prospect list. *Mentioned by Alec.*

**[Zapier](/ai-tools/zapier)** — An automation tool that connects apps and triggers actions between them, which Alec uses to push PLAUD recordings into Google Drive. *Mentioned by Alec.*

**[Make.com](https://www.make.com)** — A workflow automation tool similar to Zapier and n8n, billed per execution. *Mentioned by Alec.*

## Justin's tools

**[Hermes](/ai-tools/hermes)** — A desktop AI agent (from Nous Research) built around persistent memory that remembers everything about you, switches between models, runs cron jobs, and saves reusable skills to cut token costs, the centerpiece of Justin's segment. *Demoed by Justin.*

**[Midjourney](/ai-tools/midjourney)** — An image generation model Justin uses for creative inspiration and mock-ups (merch, cars, reimagined paintings), now expanding into hardware with a plan to replace MRI machines. *Demoed by Justin.*

**[Wispr Flow](/ai-tools/wispr-flow)** — A dictation tool you trigger with a hotkey to talk to your computer instead of typing, with real-time auto-correction so you can ramble and it cleans it up. *Demoed by Justin.*

**[Claude Design](/ai-tools/claude)** — Anthropic's design capability, which Justin rates as the best AI option for getting design truly right, used to build the Slow Hand Remodeling and AI4NTP sites from Midjourney, Figma, and Canva source files. *Demoed by Justin.*

**[GoHighLevel](/ai-tools/gohighlevel)** — A CRM with a new agentic feature, which Justin queried by voice to surface James's highest-value prospects to follow up with. *Demoed by Justin.*

**[Figma](https://www.figma.com)** — A design tool Justin uses for light wireframing and mocking up look, feel, fonts, and layout before pushing screenshots into Claude Design. *Mentioned by Justin.*

**[Canva](https://www.canva.com)** — A design tool Justin uses alongside Midjourney for mock-ups and assets feeding into Claude Design. *Mentioned by Justin.*

**[ChatGPT](/ai-tools/chatgpt)** — OpenAI's conversational assistant, which Justin used to generate a Midjourney prompt from an uploaded image, tag-teaming one AI to feed another. *Mentioned by Justin.*

**[Claude Code](/ai-tools/claude-code)** — Anthropic's coding agent that Justin calls his daily driver, used inside Visual Studio Code for getting focused work done. *Mentioned by Justin.*

**[Behance](/ai-tools/behance)** — A creative inspiration gallery Justin compared to Midjourney's explore feed for design ideas. *Mentioned by Justin.*

## Also mentioned

**Models referenced** — Inside Hermes, Justin switched between Fable 5, Opus, Sonnet, Haiku, GPT 5.5, and the cheap, open-source option **[Kimi](/ai-tools/kimi)** from Moonshot AI, plus image and video models NanoBanana (Google) and Seed Dance came up via Higgsfield. *Mentioned by Justin and Alec.*

**[Lovable](/ai-tools/lovable)** — An AI website builder Justin noted Alec likes, mentioned while comparing options for building sites. *Mentioned by Justin.*

**[OpenClaw](/ai-tools/openclaw)** — A scheduled-task AI agent in the same family as Hermes, offered as a choose-your-path alternative for the free setup boot camp (Ian hosts that one). *Mentioned by Justin and Alec.*

**[Super Whisper](https://superwhisper.com)** — A dictation tool an attendee asked about as a Wispr Flow alternative; Alec hadn't used it but said no need to switch if it's working well. *Mentioned in chat.*

**[Dartee Golf](https://darteegolf.com)** — Guest James Biernesser's golf apparel and accessories company, known for its utility golf belt with a customizable ball marker. *Mentioned by James.*

**[AI4NTP tools directory](/ai-tools)** — Our own running directory of every AI tool covered across sessions, which Justin dropped in chat for attendees who asked for the list. *Shared in chat by Justin.*
