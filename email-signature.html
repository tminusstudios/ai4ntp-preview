What changed (v2)

  1. Switched to a horizontal wordmark instead of the stacked one. The vertical "ai / 4 / ntp" stack ate too much real estate on mobile.
  2. Single column layout. No more two-column-with-divider that breaks on narrow widths.
  3. white-space: nowrap on every link, plus a non-breaking hyphen inside "Book a 30‑min". Links now wrap as whole units instead of breaking mid-word.
  4. "T Minus Studios" without the hyphen.
  5. Removed the "Latest" episode callout for evergreen use.
  6. Brand yellow (#D4FF3A) tagged with !important on the wordmark so Gmail dark mode preserves it instead of muting it to olive.
  7. The yellow underline on "Book a 30-min" pulls the eye to the primary CTA. Other links stay neutral so the visual hierarchy is clear: one yellow accent, one action.

Install in Gmail (universal method)

  1. Open this file (email-signature.html) in your browser.
  2. Click anywhere inside the top white box. Hit Cmd + A (Mac) or Ctrl + A (Windows) to select the rendered signature only.
  3. Copy: Cmd + C / Ctrl + C.
  4. In Gmail: Settings → See all settings → General → Signature → Create new.
  5. Paste with Cmd + V / Ctrl + V. Save Changes at the bottom of the page.

Install via signature tool (raw HTML upload)

  1. Open this file in any text editor.
  2. Copy everything between the SIGNATURE STARTS HERE and SIGNATURE ENDS HERE comment markers (the entire <table> block).
  3. Paste into HTMLSig, WiseStamp, or your signature manager of choice as raw HTML.

If the yellow box still looks olive on the recipient's dark mode

A small fraction of email clients (older Outlook, certain mobile clients) force-invert background colors regardless of !important. The mark stays legible in those cases (italic black "4" on a darker swatch), and brand recognition holds via the wordmark shape. Acceptable tradeoff for a text-only signature without external image hosting.
